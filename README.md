# PHP Coursework 1 - Online Bookstore

PHP coursework implementing a simple online bookstore using arrays, functions (pass-by-reference), POST forms, file logging, and HTML output.

## Features
- Book inventory management with associative arrays
- Genre-based discounts (Science Fiction: 10%, Fantasy: 5%)
- Form validation and sanitization
- File logging with timestamps, IP, and user agent
- Live log file display (bonus feature)
